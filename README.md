# Resolution-Game_Dev-Week-8

This game was made using C# and [Raylib](https://www.raylib.com/) for [Resolution Game Dev](https://resolution.hackclub.com/app/pathway/game_dev), a [Hack Club](https://hackclub.com/) YSWS.
It’s fully open source, and you’re free to use it however you like.

It's a 2D game where yoy play with a blue ball and have to dodge to type of balls blue and red ones and you get random upgrades every 30 seconds that help you survive longer.
If you wish to continue on this project you have to create a new dotnet project and download the Raylib library and paste the script into Program.ss
